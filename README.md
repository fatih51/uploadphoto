<h1>TR</h1>

Sırasıyla bu komutları yazınız.

`git clone https://github.com/fatih51/uploadphoto`

`cd server`

`mkdir uploads`

`npm install`

`nodemon index.js`

Yazıp çalıştırabilirsiniz.
